## Script Selection Guide

- **Version A (Recommended):** Data-driven restaurants (family businesses, established, professional tone)
- **Version B:** Softer approach (younger owners, casual vibe, friendly tone)
- **Version C:** Ultra-short (for testing on busy owners, low engagement accounts)

---

## Follow-Up Strategy

**If no response in 48 hours:**
1. Send Version C as a bump (shorter, re-engages)
2. Wait 5-7 days
3. Final bump: "Still interested? Happy to update the preview if anything's changed."

**If they respond "not interested":**
- Reply: "No problem! Menu stays live at [link] if you change your mind. Good luck!"
- Mark as "declined" in tracking sheet

**If they respond with questions:**
- Answer immediately
- Reference FAQ section on landing page
- Offer to hop on a quick call if needed

## Tracking Metrics

**For each DM sent, log:**
- Restaurant name
- Instagram handle
- Yelp photo count
- Menu item count
- DM version sent (A/B/C)
- Date sent
- Response? (yes/no)
- Response type (interested/questions/declined)
- Conversion? (yes/no)
- Time to conversion (hours)

**Target metrics:**
- 20% response rate (good)
- 10% conversion rate (great)
- 24-48hr response time (fast)

## Psychology Principles Used

1. **Reciprocity:** "I already built it for you"
2. **Loss aversion:** "competitors with online menus"
3. **Social proof:** "400+ Yelp photos" (implies popularity)
4. **Scarcity (subtle):** "ready to publish" (implies it could go away)
5. **Transparency:** Upfront pricing ($99)
6. **Low pressure:** "If not, no worries"
